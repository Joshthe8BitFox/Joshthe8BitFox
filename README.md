Hey, I’m Zephyr

I build weird little tools, experiments, bots, game systems, automation scripts, and custom setups mostly because I want them to exist.

A lot of my projects start as:

“This would be funny.”
“This would make my life easier.”
“I wonder if this is possible.”
“Nobody made this exactly how I wanted.”

So now my GitHub is full of personal-use utilities, hobby projects, server tools, game/modding experiments, networking configs, automation systems, and half-finished ideas that somehow became real software.

I like:

backend systems
automation
self-hosted tools
game modding
weird integrations
networking
VR and avatar tech
Linux tinkering
making unrelated things talk to each other

Most of what I make is intentionally niche, chaotic, overengineered, or highly specific to my own use cases. If any of it helps someone else too, that’s a bonus.

Not everything here is polished. Some repos are experiments. Some are prototypes. Some are “I spent 14 hours solving a problem that annoyed me once.”

But they’re fun.

If you’re here looking for enterprise-grade software… you may have taken a wrong turn.

If you’re here because you also enjoy building strange little systems for fun, welcome.
